# fsociety-terminal

Экран загрузки в стиле Mr. Robot: ASCII-логотип fsociety вращается в 3D, снизу надпись FSOCIETY, прогресс-бар - и после всего этого открывается терминал. Всё на pygame, без зависимостей кроме неё.
Писал чисто для себя, но код простой - можно взять за основу для любого похожего проекта.
## Что происходит на экране
Сначала идёт экран загрузки: ASCII-логотип (маска fsociety) вращается по оси Y в 3D-проекции. Каждый непробельный символ логотипа - это точка в трёхмерном пространстве. Вращение через матрицу rotateY, проекция перспективная (не ортогональная).
Снизу статично рендерится ASCII-надпись FSOCIETY, под ней прогресс-бар и спиннер из символов `| / - \`.
Длится 4 секунды, потом запускается терминал - просто чёрный экран, зелёный текст, ввод с клавиатуры, история команд скроллится вверх.

## Как устроено
**3D-вращение**
ASCII-логотип парсится построчно. Каждый символ, который не пробел, превращается в точку `[x, y, 0]` в трёхмерном пространстве. Координаты центрируются относительно размеров логотипа, масштабируются на `scale = 10`.
```python
def rotateY(p, angle):
    x, y, z = p
    c, s = math.cos(angle), math.sin(angle)
    return [x*c - z*s, y, x*s + z*c]

def project(p):
    x, y, z = p
    dist = 1000
    f = 800 / (z + dist)
    return int(x*f + WIDTH/2), int(y*f + HEIGHT/2 - 100)
```
Угол увеличивается на `0.05` каждый кадр - скорость вращения меняется здесь.
**Экран загрузки**
Функция `loading_animation(duration=4000)`. Принимает длительность в миллисекундах. Внутри обычный game loop: очистка экрана, рендер точек логотипа, рендер текста FSOCIETY, прогресс-бар, flip.
**Терминал**
Функция `terminal()`. Хранит список строк `lines` и текущий ввод `input_text`. Enter - добавляет строку в список, Backspace - удаляет последний символ. Вывод обрезается по высоте экрана - всегда показываются последние N строк.
## Что можно поменять
- **Скорость вращения** - `angle += 0.05` в `loading_animation`, увеличь или уменьши число
- **Длительность загрузки** - параметр `duration` при вызове `loading_animation(duration=4000)`
- **Цвет** - константа `GREEN = (0, 255, 120)`, заменяется на любой RGB
- **Размер логотипа** - переменная `scale = 10`, влияет на масштаб всей 3D-сцены
- **Шрифт** - `logo_font`, `text_font`, `term_font` в начале файла
- **Логотип** - список `ascii_logo`, можно вставить любой ASCII-арт
- **Надпись снизу** - список `fs_text`
